This page lists and describes the java files in your elevatorcontrol package.  Update this document to reflect your implementation and add your java files to the elevatorcontrol folder.

Note:  In order for your portfolio to be complete, the code files in the elevatorcontrol folder must compiled and run if copied into the elevatorcontrol package (overwriting any duplicates) of a clean version of the latest required simulator release.

Controller files

  • CarButtonControl.java - A class controls car buttons
  • CarPositionControl.java - A class indicates car position
  • Dispatcher.java - A class computes next desired floor
  • DoorControl.java - A class controls doors
  • DriveControl.java - A class issues drive command to elevator
  • HallButtonControl.java - A class controls hall buttons
  • LanternControl.java - A class controls latterns

Translator files

Links and descriptions to all message translators used by the controllers.
  • CarCallCanPayloadTranslator.java - A class translates car call network messages
  • DesiredFloorCanPayloadTranslator.java - A class translates desired floor network messages
  • DriveCommandCanPayloadTranslator.java - A class translates drive command network messages
  • DriveSpeedCanPayloadTranslator.java - A class translates drive speed network messages
  • HallCallCanPayloadTranslator.java - A class translates hall call network messages

Utility files

  • Utility.java - A class contains AtFloorArray, DoorClosedArray, HallCallArray and CarCallArray
  • MessageDictionary.java - A class defines network message IDs
  • Proj7RuntimeMonitor.java - A class implements a monitor for project 7
  • SampleDispatcherMonitor.java - Sample monitor for dispatcher
  • SamplePerformanceMonitor.java - Sample monitor for elevator performance
  • RuntimeRequirementsMonitor.java - Runtime monitor for high level requirements verification
  • TestLight.java - A sample class for controller class implementation
